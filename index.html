How to play free Gamaxy games
1º To run Gamaxy game applets please download the latest version of Sun's J2SE 1.4.2 JRE, and follow the Sun's installation instructions.  
2º Select a game you would like to play and the following window appears.  
     
2ºa' New Game link allows you to initiate a new game by providing:
  • Your alias
  • Game name
  • Game parameters
  
2ºb' <game name> - a green link allows you to join a game as a player by providing:
  • Your alias
  
2ºc' <game name> - red link allows you to join a game as a spectator by providing:
  • Your alias
  
3º Applets are signed by Gamaxy, you need to accept the certificate by selecting "Yes" or "Always".  
     
4º Once required number of players is logged on the game starts, e.g. Preference will start once 3 players are logged.
